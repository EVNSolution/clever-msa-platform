# front-admin-console

이 repo는 관리자 front UI를 소유한다.

현재 역할:
- master/assignment 관리 화면
- admin 계정 기준 CRUD 화면
- admin-only API client와 router

포함:
- React/Vite app
- admin page test
- admin route와 API client

포함하지 않음:
- operator UI
- gateway 설정
- backend 도메인 로직

- `../../docs/contracts/`
- `../../docs/boundaries/`
