# front-driver-app

`front-driver-app` is the CLEVER driver-only mobile app repo.

Current product target:

- framework: `React Native + Expo`
- operation: `native-only`
- tenant phase-1 target: `cheonha`
- current scope: `login`, `signup`, `work-logs`, `my`, `admin-empty`

## Local Commands

```sh
npm install
npm run typecheck
npm run ios
npm run android
```

Notes:

- `npm run ios` maps to `expo run:ios`
- `npm run android` maps to `expo run:android`
- web is not the product target for this repo
